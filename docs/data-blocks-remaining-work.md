## 二、尚未实现的能力

### 2.1 时间范围切换功能

**状态**: 未实现

#### 当前情况
- 趋势类积木目前基本是固定时间范围
- 前端没有时间范围切换控件
- 新增卡片时也不能配置时间范围

#### 结论
需要支持 `24h / 7天 / 30天 / 自定义` 等时间范围切换。
