## 3. 详细工作内容

### 3.1 定义 Role 领域对象

创建 `src/domain/role.ts`：

```typescript
export type RoleId = string;

export interface Role {
  id: RoleId;
  name: string;                    // 角色名称（如 "专业客服"、"友好助手"）
  description: string;             // 角色描述
  prompt: string;                  // 角色提示词（用于注入到 AI prompt）
  workspaceId: string;             // 归属客户空间
  createdAt: Date;
  metadata?: Record<string, unknown>;
}

export interface CreateRoleParams {
  name: string;
  description: string;
  prompt: string;
  workspaceId: string;
  metadata?: Record<string, unknown>;
}
```

### 3.2 定义 AccountRole 绑定关系

创建 `src/domain/accountRole.ts`：

```typescript
export interface AccountRole {
  accountId: string;
  roleId: string;
  isDefault: boolean;              // 是否为默认角色
  createdAt: Date;
}
```

### 3.3 实现 Role 仓储

创建 `src/data/repositories/IRoleRepository.ts`：

```typescript
export interface IRoleRepository {
  save(role: Role): Promise<void>;
  findById(id: RoleId): Promise<Role | null>;
  findByWorkspace(workspaceId: string): Promise<Role[]>;
  delete(id: RoleId): Promise<void>;
  
  // 账号-角色绑定
  bindRole(accountId: string, roleId: string, isDefault: boolean): Promise<void>;
  unbindRole(accountId: string, roleId: string): Promise<void>;
  getAccountRoles(accountId: string): Promise<Role[]>;
  getDefaultRole(accountId: string): Promise<Role | null>;
}
```

创建 `src/data/repositories/InMemoryRoleRepository.ts`：

```typescript
export class InMemoryRoleRepository implements IRoleRepository {
  private roles: Map<RoleId, Role> = new Map();
  private accountRoles: Map<string, AccountRole[]> = new Map();
  
  // 实现所有接口方法
}
```

### 3.4 实现角色管理 UI

创建 `src/features/roles/` 目录，包含：
