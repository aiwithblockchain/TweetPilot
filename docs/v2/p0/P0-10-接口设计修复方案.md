## 问题描述

### 当前架构违规

在阶段2（LocalBridge集成）实现过程中，违反了 [P0-03-接口设计规范.md](P0-03-接口设计规范.md) 的核心原则：

**违规点1：实现细节泄漏到接口层**

```typescript
// src/services/account/types.ts:40-50
export interface AccountService {
  // ... 其他方法
  syncAccountsFromLocalBridge(): Promise<MappedAccount[]>  // ❌ 暴露实现细节
  getLocalBridgeConfig(): Promise<{...}>                   // ❌ 配置管理职责错位
  updateLocalBridgeConfig(...): Promise<void>              // ❌ 配置管理职责错位
}
```

**违规点2：违反单一职责原则**

- `AccountService` 应该只负责账号映射和状态管理
- LocalBridge 是**数据源实现细节**，不应暴露给接口层
- 配置管理属于 `SettingsService` 的职责

**违规点3：Mock 实现被迫添加空实现**

```typescript
// src/services/account/mock.ts:200-216
async syncAccountsFromLocalBridge() {
  await randomDelay(500, 1000)
  return []  // ❌ 无意义的空实现
}
```

---

## 设计原则回顾

根据 [P0-03-接口设计规范.md](P0-03-接口设计规范.md) 第2.1节：

> UI 只能依赖稳定的前端 service interface。
> 接口由 UI 和场景驱动，但 UI 不直接依赖：
> - 某个 Mock 类
> - 某个具体 Tauri Command 名称
> - 某个临时数据结构

**核心问题**：LocalBridge 是 Tauri 实现的数据源，不应该成为接口的一部分。

---

## 修复方案

### 方案概述

1. **从 `AccountService` 接口删除3个 LocalBridge 方法**
2. **将配置管理移到 `SettingsService`**
3. **同步逻辑改为后台任务，不暴露给前端**
4. **UI 的"同步按钮"改为触发通用的 `refreshAllAccountsStatus()`**

### 架构调整

#### 修改前（违规）

```
UI (AccountManagement)
  ↓
accountService.syncAccountsFromLocalBridge()  ← 暴露实现细节
  ↓
Tauri Command: sync_accounts_from_localbridge
  ↓
LocalBridge HTTP Client
```

#### 修改后（符合规范）

```
UI (AccountManagement)
  ↓
accountService.refreshAllAccountsStatus()  ← 通用接口
  ↓
Tauri 实现内部决定数据源（LocalBridge/Mock/其他）
  ↓
后台任务自动同步（用户无感知）
```

```rust
#[tauri::command]
pub async fn refresh_all_accounts_status() -> Result<(), String> {
    // 内部实现：从 LocalBridge 同步账号状态
    let config = storage::load_localbridge_config()?;
    let client = LocalBridgeClient::new(&config.endpoint, config.timeout_ms)?;
    
    match client.get_instances().await {
        Ok(instances) => {
            // 更新账号状态到本地存储
            for instance in instances {
                // ... 更新逻辑
            }
            Ok(())
        }
        Err(e) => {
            eprintln!("LocalBridge 同步失败: {}", e);
            // 不抛错，降级为本地状态刷新
            Ok(())
        }
    }
}
```
