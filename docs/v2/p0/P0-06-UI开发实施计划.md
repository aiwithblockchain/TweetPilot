## 2. 技术栈

### 2.1 前端技术栈
- **框架**：Tauri 2.x + Vite + React 19 + TypeScript
- **状态管理**：Zustand（轻量级）
- **UI 组件**：shadcn/ui（基于 Radix + Tailwind CSS）
- **图表库**：Recharts（轻量级）
- **拖拽库**：@dnd-kit/core
- **国际化**：react-i18next
- **图标**：Lucide React

### 2.2 后端技术栈（Rust）
- **框架**：Tauri 2.x
- **运行时**：Tokio（异步运行时）
- **进程通信**：Tauri Commands
- **文件存储**：本地 JSON 文件（serde_json）
- **定时任务**：tokio + cron 库
- **子进程**：tauri-plugin-shell（执行 Python 脚本）
- **对话框**：tauri-plugin-dialog

### 2.3 设计规范
- **色彩体系**：参考 DESIGN-02-色彩体系.md
  - Brand primary: `#6D5BF6`
  - Dark theme (default): `--color-bg: #0F1117`, `--color-text: #E8E9ED`
- **排版**：参考 DESIGN-03-排版与间距.md
  - Font: Inter (UI), JetBrains Mono (code)
  - Base font size: 14px
  - Spacing: 4px grid system
- **组件**：参考 DESIGN-04-组件规范.md
- **布局**：参考 DESIGN-05-页面布局.md

```typescript
}

export const useWorkspaceStore = create<WorkspaceStore>((set) => ({
  currentWorkspace: null,
  recentWorkspaces: [],
  
  setCurrentWorkspace: async (path) => {
    await TauriService.setCurrentWorkspace(path)
    set({ currentWorkspace: path })
  },
  
  loadRecentWorkspaces: async () => {
    const workspaces = await TauriService.getRecentWorkspaces()
    set({ recentWorkspaces: workspaces })
  },
}))
```
