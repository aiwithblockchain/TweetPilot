### 阶段 1：GitHub 克隆功能（优先级：高）

**目标**：实现真实的 `git clone` 调用。

#### 任务清单

1. **实现 Git 命令检测**
   - 检查系统是否安装 Git
   - 返回清晰的错误信息

2. **实现 URL 验证**
   - 验证 GitHub URL 格式
   - 支持 HTTPS 和 SSH 格式
   - 示例：
     - `https://github.com/owner/repo.git`
     - `git@github.com:owner/repo.git`

3. **实现克隆逻辑**
   - 验证目标目录不存在或为空
   - 调用 `git clone` 命令
   - 捕获命令输出和错误
   - 克隆成功后设置为当前工作区

4. **错误处理**
   - Git 未安装
   - 网络错误
   - 认证失败
   - 目录已存在
   - 仓库不存在

#### 技术实现

```rust
use std::process::Command;
use std::path::Path;

pub async fn clone_from_github(repo_url: String, target_dir: String) -> Result<String, String> {
    // 1. 检查 Git 是否安装
    let git_check = Command::new("git")
        .arg("--version")
        .output();
    
    if git_check.is_err() {
        return Err("Git 未安装，请先安装 Git".to_string());
    }

    // 2. 验证 URL 格式
    if !is_valid_github_url(&repo_url) {
        return Err("无效的 GitHub URL".to_string());
    }

    // 3. 验证目标目录
    if Path::new(&target_dir).exists() {
        return Err("目标目录已存在".to_string());
    }

    // 4. 执行克隆
    let output = Command::new("git")
        .arg("clone")
        .arg(&repo_url)
        .arg(&target_dir)
        .output()
        .map_err(|e| format!("克隆失败: {}", e))?;

    if !output.status.success() {
        let error = String::from_utf8_lossy(&output.stderr);
        return Err(format!("克隆失败: {}", error));
    }

    // 5. 设置为当前工作区
    set_current_workspace(target_dir.clone()).await?;

    Ok(target_dir)
}

fn is_valid_github_url(url: &str) -> bool {
    url.starts_with("https://github.com/") || 
    url.starts_with("git@github.com:")
}
```

#### LocalBridge API 设计

假设 LocalBridge 提供以下 API：

```
GET  /api/accounts/{screenName}/status
POST /api/accounts/{screenName}/reconnect
GET  /api/accounts/{screenName}/profile
```

#### 技术实现

```rust
use reqwest::Client;
use serde::{Deserialize, Serialize};

#[derive(Debug, Serialize, Deserialize)]
struct LocalBridgeConfig {
    endpoint: String,
    api_key: String,
    timeout_ms: u64,
}

#[derive(Debug, Serialize, Deserialize)]
struct AccountStatusResponse {
    status: String, // "online" | "offline" | "error"
    last_active: Option<String>,
}

pub async fn verify_account_status(screen_name: String) -> Result<String, String> {
    // 1. 读取配置
    let config = load_local_bridge_config()?;

    // 2. 创建 HTTP 客户端
    let client = Client::builder()
        .timeout(std::time::Duration::from_millis(config.timeout_ms))
        .build()
        .map_err(|e| format!("创建 HTTP 客户端失败: {}", e))?;

    // 3. 发送请求
    let url = format!("{}/api/accounts/{}/status", config.endpoint, screen_name);
    let response = client
        .get(&url)
        .header("Authorization", format!("Bearer {}", config.api_key))
        .send()
        .await
        .map_err(|e| format!("请求失败: {}", e))?;

    // 4. 解析响应
    if !response.status().is_success() {
        return Err(format!("API 返回错误: {}", response.status()));
    }

    let status_response: AccountStatusResponse = response
        .json()
        .await
        .map_err(|e| format!("解析响应失败: {}", e))?;

    Ok(status_response.status)
}

pub async fn reconnect_account(screen_name: String) -> Result<(), String> {
    let config = load_local_bridge_config()?;
    let client = Client::builder()
        .timeout(std::time::Duration::from_millis(config.timeout_ms))
        .build()
        .map_err(|e| format!("创建 HTTP 客户端失败: {}", e))?;

    let url = format!("{}/api/accounts/{}/reconnect", config.endpoint, screen_name);
    let response = client
        .post(&url)
        .header("Authorization", format!("Bearer {}", config.api_key))
        .send()
        .await
        .map_err(|e| format!("请求失败: {}", e))?;

    if !response.status().is_success() {
        return Err(format!("重连失败: {}", response.status()));
    }

    Ok(())
}
```

#### 技术实现

```rust
use std::process::Command;
use std::time::Instant;

#[derive(Debug, Serialize, Deserialize)]
struct ExecutionResult {
    status: String, // "success" | "failure"
    duration: f64,  // seconds
    output: String,
    error: Option<String>,
    timestamp: String,
}

pub async fn execute_task(task_id: String) -> Result<ExecutionResult, String> {
    let start = Instant::now();
    let timestamp = chrono::Utc::now().to_rfc3339();

    // 1. 获取任务信息
    let task = get_task(&task_id)?;

    // 2. 验证脚本文件
    let script_path = Path::new(&task.script_path);
    if !script_path.exists() {
        return Err(format!("脚本文件不存在: {}", task.script_path));
    }

    // 3. 确定脚本类型和执行器
    let (executor, args) = get_executor_for_script(&task.script_path)?;

    // 4. 执行脚本
    let output = Command::new(executor)
        .args(&args)
        .arg(&task.script_path)
        .current_dir(&task.working_directory)
        .output()
        .map_err(|e| format!("执行失败: {}", e))?;

    // 5. 计算执行时间
    let duration = start.elapsed().as_secs_f64();

    // 6. 构建结果
    let result = ExecutionResult {
        status: if output.status.success() {
            "success".to_string()
        } else {
            "failure".to_string()
        },
        duration,
        output: String::from_utf8_lossy(&output.stdout).to_string(),
        error: if output.status.success() {
            None
        } else {
            Some(String::from_utf8_lossy(&output.stderr).to_string())
        },
        timestamp,
    };

    // 7. 保存执行历史
    save_execution_history(&task_id, &result)?;

    // 8. 更新任务统计
    update_task_statistics(&task_id, &result)?;

    Ok(result)
}

fn get_executor_for_script(script_path: &str) -> Result<(String, Vec<String>), String> {
    if script_path.ends_with(".sh") {
        Ok(("sh".to_string(), vec![]))
    } else if script_path.ends_with(".py") {
        Ok(("python3".to_string(), vec![]))
    } else if script_path.ends_with(".js") {
        Ok(("node".to_string(), vec![]))
    } else {
        Err("不支持的脚本类型".to_string())
    }
}
```

#### 安全考虑

1. **路径验证**
   - 脚本路径必须在工作区内
   - 禁止访问系统敏感目录
   - 禁止使用 `..` 路径遍历

2. **执行权限**
   - 脚本必须有可执行权限
   - 不自动添加执行权限

3. **超时控制**
   - 设置默认超时（如 5 分钟）
   - 超时后终止进程

4. **资源限制**
   - 限制同时执行的任务数量
   - 防止资源耗尽

- **风险**：网络不稳定

- **风险**：恶意脚本

- **风险**：脚本执行时间过长
