## 技术实现

### 存储服务

所有持久化通过统一的 storage 服务实现：

**位置**：[src-tauri/src/services/storage.rs](../../../src-tauri/src/services/storage.rs)

**功能**：
- `read_json<T>()` - 读取 JSON 文件，文件不存在时返回默认值
- `write_json<T>()` - 原子写入 JSON 文件（写临时文件 → 重命名）
- 自动创建 `~/.tweetpilot/` 目录
- 错误处理：文件不存在不报错，格式错误返回默认值

### 实现模式

**模式 A：完全无状态持久化**（Workspace, Preferences, Account, DataBlocks）
- 每次操作都读取文件
- 修改后立即写入文件
- 无内存状态，重启安全

**模式 B：混合模式**（Task）
- 启动时加载到内存（`Lazy<Mutex<T>>`）
- 修改后调用 `persist_*()` 保存到文件
- 保持内存状态以提高性能
