## 1. 全局布局

### 1.1 结构

```
┌──────────────────────────────────────────────┐
│ Header (48px)                                │
├──────────┬───────────────────────────────────┤
│ Sidebar  │ Content Area                      │
│ (220px)  │ (flex: 1)                         │
│          │                                   │
│          │                                   │
│          │                                   │
│          │                                   │
│          │                                   │
└──────────┴───────────────────────────────────┘
```

- 全屏：`width: 100vw; height: 100vh; overflow: hidden`
- 背景：`var(--color-bg)`
- 最小窗口宽度：960px（低于此宽度进入窄模式，侧边栏折叠为图标）

### 1.2 Header

```
┌──────────────────────────────────────────────┐
│ [Logo] TweetPilot  │  /workspace/path  │ [btn] [btn] │
└──────────────────────────────────────────────┘
```

- 高度：48px（原型 60px，降低以增加内容区空间）
- 背景：`var(--color-bg-elevated)`
- 底边框：`1px solid var(--color-border)`
- 左侧：Logo 图标（24px）+ 名称（`--text-base, --font-semibold`）
- 中间：当前工作目录路径（`--text-sm, --color-text-secondary`，截断显示）
- 右侧：操作按钮组（打开目录、设置）
- Padding：`0 var(--space-4)`

### 1.3 Sidebar

见 DESIGN-04 组件规范第 5 节。

### 1.4 Content Area

- Padding：`var(--space-6)`（24px）
- 背景与全局一致：`var(--color-bg)`
- 滚动：内容区独立滚动，`overflow-y: auto`

## 2. 任务管理页

### 2.1 页面结构

```
┌───────────────────────────────────────────┐
│ 任务管理                    [筛选] [+创建] │  ← page-header
├──────────┬────────────────────────────────┤
│          │                                │
│  任务列表 │       任务详情 / 空状态         │  ← 左列表 + 右详情
│  (280px) │       (flex: 1)                │
│          │                                │
└──────────┴────────────────────────────────┘
```

### 2.2 Page Header

- 左侧：页面标题 "任务管理"（`--text-xl, --font-semibold`）
- 右侧：筛选下拉 + "创建任务" Primary 按钮
- 高度自适应，底部 `margin-bottom: var(--space-4)`

### 2.3 任务列表（左栏）

- 宽度：280px
- 右边框：`1px solid var(--color-border)`
- 顶部：搜索框（Filter input）
- 列表项：

```
┌─────────────────────────┐
│ ● 任务名称              │  ← --font-medium, 单行截断
│   类型 · 下次执行时间    │  ← --text-xs, --color-text-secondary
└─────────────────────────┘
```

- 列表项高度：52px，padding `var(--space-2) var(--space-3)`
- 选中态：背景 `var(--color-primary-50)`（Dark: `--primary-800`），左边条 2px `--color-primary`
- Hover：背景 `var(--color-bg-hover)`

### 2.4 任务详情（右栏）

无选中任务时显示 Empty State："选择一个任务查看详情"。

选中任务后：

```
┌────────────────────────────────────┐
│ 任务名称                   [操作▼]  │  ← 标题 + 更多操作菜单
│ ────────────────────────────────── │
│                                    │
│ [基本信息区]                        │
│ 类型: 即时任务                      │
│ 脚本: /scripts/example.js          │
│ 创建: 2026-04-15                   │
│                                    │
│ [参数配置区]                        │
│ 参数表单...                         │
│                                    │
│ [操作按钮区]                        │
│ [执行] [暂停] [删除]                │
│                                    │
│ [执行历史] (仅定时任务)              │
│ 时间   状态  耗时                   │
│ ...                                │
└────────────────────────────────────┘
```

- 内容区间隔：`var(--space-6)`
- 子区域标题：`--text-base, --font-medium, --color-text-secondary`，下方 `margin-bottom: var(--space-3)`

### 2.5 创建任务模态框

- 标题："创建新任务"
- 表单字段：
  1. 任务名称（Text input，必填）
  2. 任务类型（Select: 即时任务 / 定时任务）
  3. 脚本路径（File picker 按钮 + 路径显示）
  4. 执行计划（Text input，仅定时任务时显示，格式提示 "every N minutes/hours/days"）
  5. 参数配置（Key-Value 动态表单，可增减行）
- 底部按钮：[取消] [创建]

## 3. 数据积木页

### 3.1 页面结构

```
┌───────────────────────────────────────────┐
│ 数据积木           [账号选择▼] [+添加卡片] │  ← page-header
├───────────────────────────────────────────┤
│                                           │
│  ┌──────┐ ┌──────┐ ┌──────┐              │
│  │卡片 1│ │卡片 2│ │卡片 3│              │  ← 卡片网格 (3列)
│  └──────┘ └──────┘ └──────┘              │
│  ┌──────┐ ┌──────┐                       │
│  │卡片 4│ │卡片 5│                       │
│  └──────┘ └──────┘                       │
│                                           │
└───────────────────────────────────────────┘
```

### 3.2 Page Header

- 左侧：页面标题 "数据积木"
- 右侧：账号选择器 Dropdown + "添加卡片" Primary 按钮
- 账号选择器：头像（20px）+ 名称 + 下拉箭头

### 3.3 卡片网格

- 列数：默认 3 列，`grid-template-columns: repeat(3, 1fr)`
- 响应式断点：
  - 内容区宽度 < 800px：2 列
  - 内容区宽度 < 560px：1 列
- 列间距：`gap: var(--space-4)`
- 行间距：`gap: var(--space-4)`

### 3.4 卡片内容类型

每种数据卡片的内容布局：

**粉丝统计（account_basic_data）**：

```
┌────────────────────┐
│ 1.2K    2.5M       │  ← 2x2 网格, 数字 --text-3xl, --font-bold
│ 关注     被关注     │     标签 --text-xs, --color-text-secondary
│                    │
│ 850      3,240     │
│ 蓝V关注   总推文    │
└────────────────────┘
```

**推文互动数据（account_interaction_data）**：

```
┌────────────────────┐
│ 1.2B  48M   12M    │  ← 3 列, 大数字 + 小标签
│ 浏览   点赞   转推  │
└────────────────────┘
```

**最新推文列表（latest_tweets）**：

```
┌────────────────────┐
│ 推文内容文字...      │  ← --text-sm, 最多 2 行截断
│ 👁 1.2K ❤️ 856 💬 12 │  ← 互动数据 --text-xs
│ 3分钟前             │  ← --text-xs, --color-text-tertiary
├────────────────────┤
│ 推文内容文字...      │
│ 👁 980  ❤️ 234 💬 5  │
│ 1小时前             │
└────────────────────┘
```

- 列表项之间：`border-bottom: 1px solid var(--color-border)`
- 最多显示 5 条，超出隐藏

**推文时间分布（tweet_time_distribution）**：

- 柱状图，使用 Recharts BarChart
- 图表高度 200px
- 网格线颜色参考 DESIGN-02 图表色

**任务执行统计（task_execution_stats）**：

- 环形图（Donut chart），使用 Recharts PieChart
- 中心显示成功率百分比
- 图表高度 200px
- 底部图例：成功（绿）+ 失败（红）

### 3.5 添加卡片模态框

- 标题："添加数据卡片"
- 内容：卡片类型选择网格（2 列）
- 每个选项：卡片容器，包含标题 + 描述
- Hover：`border-color: var(--color-primary)`，`transform: translateY(-2px)`
- 点击即选中并关闭模态框

## 4. 设置页

### 4.1 页面结构

```
┌───────────────────────────────────────────┐
│ 设置                                      │  ← page-header
├──────────┬────────────────────────────────┤
│          │                                │
│  设置导航 │       设置内容                  │
│ (160px)  │       (flex: 1)                │
│          │                                │
└──────────┴────────────────────────────────┘
```

### 4.2 设置导航（页内左侧导航）

- 通用设置
- 账号管理
- 关于

选中态与全局侧边栏导航一致：背景色 + 左边条。

### 4.3 通用设置

表单布局：

```
外观
├── 主题     [Dark / Light / System]    ← Segmented control
├── 语言     [简体中文 ▼]
└── 启动     [ ] 开机自启动

通知
├── 任务执行通知  [toggle]
└── 错误告警      [toggle]
```

- 分组标题：`--text-base, --font-medium`，下方 `margin-bottom: var(--space-3)`
- 分组间距：`var(--space-8)`
- 每个设置项：Label + 控件 水平排列，`justify-content: space-between`

### 4.4 账号管理

列表布局，每行一个账号：

```
┌────────────────────────────────────────┐
│ [Avatar] 名称  @handle    ● 在线  [删除] │
│          上次同步: 2分钟前               │
└────────────────────────────────────────┘
```

- 账号卡片之间：`gap: var(--space-2)`
- 底部："+ 绑定新账号" Ghost 按钮

### 4.5 关于

- Logo（64px）+ 应用名称 + 版本号
- 品牌渐变分割线
- 链接列表：GitHub / 文档 / 反馈
- 简短描述文字

## 5. 响应式策略

### 5.1 断点

| 断点 | 内容区宽度 | 变化 |
|------|-----------|------|
| 宽屏 | > 1200px | 侧边栏 220px，数据积木 3 列 |
| 标准 | 960-1200px | 侧边栏 220px，数据积木 2 列 |
| 窄屏 | < 960px | 侧边栏折叠为图标栏（48px），数据积木 1-2 列 |

### 5.2 窄屏模式

- 侧边栏仅显示图标，无文字，宽度 48px
- 任务管理页：列表/详情切换显示，不再并排
- 数据积木：网格列数减少
- Header 中间的工作目录路径隐藏，改为 tooltip
