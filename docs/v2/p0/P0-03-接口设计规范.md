# P0 阶段 - 接口设计规范（Tauri Commands）

## 文档信息
- 版本：v2.0.0（更新为 Tauri 架构）
- 创建日期：2026-04-13
- 最后更新：2026-04-15

- 负责人：技术团队

## 1. 文档说明

本文档描述 TweetPilot V2 P0 阶段所有 Tauri Commands 的设计规范，建立 React 前端与 Rust 后端之间的接口映射关系。

**核心原则**:
- 接口由 UI 需求驱动，而非预先设计
- 每个 Command 都有明确的 UI 对应
- Rust 类型定义先于实现，Mock 先于真实实现
- 接口设计遵循单一职责原则

**P0 阶段接口范围**:
- 工作目录管理 Commands (workspace.rs)
- 账号管理 Commands (account.rs)
- 任务管理 Commands (task.rs)
- 数据积木 Commands (data_blocks.rs)
- LocalBridge 通信 (bridge.rs)

## 2. 接口设计流程

### 2.1 设计流程图

```
UI 界面实现（硬编码数据）
    ↓
分析 UI 需求
    ↓
定义 Rust Tauri Commands
    ↓
创建 Mock 实现（返回假数据）
    ↓
前端通过 invoke 调用 Mock Commands
    ↓
开发真实 Rust 实现
    ↓
替换 Mock 为真实实现
```

### 2.2 命名规范

**Rust 文件位置**: `src-tauri/src/commands/`

**命名规则**:
- Command 函数使用 snake_case 命名
- 文件名与功能模块对应
- 使用 `#[tauri::command]` 宏标注

**示例**:
```
src-tauri/src/commands/workspace.rs  → select_local_directory, get_current_workspace
src-tauri/src/commands/task.rs       → create_task, execute_task
src-tauri/src/commands/account.rs    → map_account, get_mapped_accounts
```

### 2.3 数据模型位置

**Rust 数据模型位置**: `src-tauri/src/models/`

**命名规则**:
- 使用 PascalCase 命名
- 每个实体一个文件
- 使用 `#[derive(Serialize, Deserialize)]` 支持 JSON 序列化

**前端类型定义位置**: `src/types/`

## 7. Mock 实现规范

### 7.1 Mock 文件位置

**Mock 实现文件位置**: `src/mocks/`

**命名规则**:
- Mock 类名以 `Mock` 开头
- 文件名与接口名对应(去掉 `I` 前缀,加 `Mock` 前缀)

**示例**:
```
src/mocks/MockProductManagement.ts  → export class MockProductManagement
src/mocks/MockTaskManagement.ts     → export class MockTaskManagement
src/mocks/MockDataStorage.ts        → export class MockDataStorage
```

### 7.2 Mock 实现要求

1. **完全符合接口定义**: Mock 类必须实现接口的所有方法
2. **返回符合规范的数据**: 返回的数据必须符合数据模型定义
3. **模拟真实行为**: 包括延迟、错误处理等
4. **可配置**: 支持配置返回的数据和行为

### 7.3 Mock 实现示例

```typescript
// src/mocks/MockProductManagement.ts
import { IProductManagement } from '../interfaces/ProductManagement';
import { Product, CreateProductDTO, UpdateProductDTO } from '../types/Product';

export class MockProductManagement implements IProductManagement {
  private products: Map<string, Product> = new Map();

  constructor() {
    // 初始化一些假数据
    this.initMockData();
  }

  private initMockData(): void {
    const mockProduct: Product = {
      id: 'product-1',
      name: '我的 SaaS 产品',
      description: '一个很棒的 SaaS 产品',
      twitterAccountId: 'account-1',
      createdAt: new Date('2026-01-01'),
      updatedAt: new Date('2026-04-13'),
      lastUsedAt: new Date('2026-04-13'),
    };
    this.products.set(mockProduct.id, mockProduct);
  }

  async listProducts(): Promise<Product[]> {
    // 模拟网络延迟
    await this.delay(300);
    return Array.from(this.products.values());
  }

  async searchProducts(keyword: string): Promise<Product[]> {
    await this.delay(200);
    const allProducts = Array.from(this.products.values());
    return allProducts.filter(p =>
      p.name.toLowerCase().includes(keyword.toLowerCase())
    );
  }

  async getProduct(productId: string): Promise<Product> {
    await this.delay(100);
    const product = this.products.get(productId);
    if (!product) {
      throw new Error(`Product ${productId} not found`);
    }
    return product;
  }

  async createProduct(data: CreateProductDTO): Promise<Product> {
    await this.delay(500);
    const newProduct: Product = {
      id: `product-${Date.now()}`,
      ...data,
      createdAt: new Date(),
      updatedAt: new Date(),
      lastUsedAt: new Date(),
    };
    this.products.set(newProduct.id, newProduct);
    return newProduct;
  }

  async updateProduct(
    productId: string,
    data: UpdateProductDTO
  ): Promise<Product> {
    await this.delay(300);
    const product = this.products.get(productId);
    if (!product) {
      throw new Error(`Product ${productId} not found`);
    }
    const updatedProduct = {
      ...product,
      ...data,
      updatedAt: new Date(),
    };
    this.products.set(productId, updatedProduct);
    return updatedProduct;
  }

  async deleteProduct(productId: string): Promise<void> {
    await this.delay(300);
    if (!this.products.has(productId)) {
      throw new Error(`Product ${productId} not found`);
    }
    this.products.delete(productId);
  }

  async selectProduct(productId: string): Promise<void> {
    await this.delay(100);
    const product = this.products.get(productId);
    if (!product) {
      throw new Error(`Product ${productId} not found`);
    }
    product.lastUsedAt = new Date();
    this.products.set(productId, product);
  }

  async getCurrentProduct(): Promise<Product | null> {
    await this.delay(100);
    // 返回最近使用的产品
    const products = Array.from(this.products.values());
    if (products.length === 0) return null;
    return products.sort((a, b) =>
      b.lastUsedAt.getTime() - a.lastUsedAt.getTime()
    )[0];
  }

  private delay(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}
```
